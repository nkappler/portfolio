Blender Renders

Pixel Art

These pixel art animations were made by creating a 3D model in blender, rendering it with very low resolution and framerate.
The result was then upscaled and a custom color palette with only a few colors was applied with gimp.

Video games

Centipede

This is a university project that I've created back in 2016 with C++ and OpenGL. It is a (very bad) clone of the 1980s Arcade Game.

The game is very incomplete: There are neither points, nor highscores, the only enemy is the centipede and you only have one life. There are no levels and the mushrooms don't change colors like in the original, but at least there's an end screen (made of mushrooms).

You can download it here

Bacteria Raid

Bacteria Raid is another university project. We have created this game in a group of three.
You are in control of a bacteria infiltrating the body. You need to reach the end of the level (the blood vessel) without dying.
The game was made with Unreal Engine, the Models were created with Blender and the level is procedurally generated.
